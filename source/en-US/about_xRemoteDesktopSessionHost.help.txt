TOPIC
    about_xRemoteDesktopSessionHost

SHORT DESCRIPTION
    DSC resources to manage Remote Desktop Services Hosts

LONG DESCRIPTION
    This module is meant to enable DSC configured RDSH (Remote Desktop Session Hosts).
    It allows for setting up a connection broker, configuring licensing and deployment
    of remote applications.

EXAMPLES
    See the examples in the Examples folder.
